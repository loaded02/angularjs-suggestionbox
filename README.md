# AngularJS-Suggestionbox

##dependencies:
* git
* node.js
* npm
* bower
* mongodb

##get started:
* clone repository
* cd rep folder
* npm install
* bower install
* service mongod restart
* node server
* visit http://localhost:3000/ in browser
